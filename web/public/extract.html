知識提取

先選擇對話來源工具，再用對應入口載入 Copilot / ChatGPT / Gemini / Claude / 純文字內容，系統會以正確 adapter 正規化為 `ConversationDoc`，再進入既有提取、審核與寫回流程。

1 選擇來源並載入對話
工具來源

選擇要匯入的工具來源；每個來源面板都會標示「支援格式」與目前限制。

VS Code Copilot 本機匯入

從本機 `Code / Code - Insiders` 的 Copilot Chat session JSONL 讀取最近對話，不需手動複製貼上。

folder 尚未讀取 Copilot session 清單。
info 支援格式：本機 VS Code Copilot Chat session JSONL。限制：需先從清單載入單筆 session，不能直接貼上 transcript。

ChatGPT 匯入

支援貼上分享頁 transcript、上傳官方 conversation JSON / TXT，或透過 OpenAI API 載入本工作台追蹤的 conversations。

info 支援格式：ChatGPT 分享頁 transcript、官方 conversation JSON / TXT，或已追蹤的 OpenAI platform conversation API session。限制：API 載入不會直接列出 ChatGPT 產品聊天歷史，需先設定 API key 並追蹤 `conversationId`。
forum 可直接貼上 transcript，或先上傳檔案再提取候選知識。

ChatGPT API 載入

先在 設定 儲存 OpenAI API key，再追蹤 `conversationId` 並從 tracked sessions 載入。

cloud 尚未檢查 OpenAI API 設定狀態。

Gemini 匯入

支援貼上 Gemini 對話 transcript；系統會明確走 Gemini adapter，保留 `gemini` 來源標記。

star 支援格式：Gemini 網頁複製的 transcript 全文，或包含 `You` / `Gemini` turns 的貼上文字。限制：本輪不支援 API 載入、檔案上傳或多 conversation picker。

Claude 匯入

支援貼上 Claude 對話 transcript；系統會明確走 Claude adapter，保留 `claude` 來源標記。

psychology 支援格式：Claude.ai 網頁複製的 transcript 全文，或包含 `Human` / `Assistant` / `Claude` 標頭的貼上文字。限制：本輪不支援 API 載入、檔案上傳或多 conversation picker。

純文字匯入

貼上一般對話或整理過的文字內容，系統會以 plain text adapter 進入既有 extraction 流程。

notes 支援格式：一般對話或人工整理後的純文字貼上。限制：這個模式不做 ChatGPT / Gemini / Claude / Copilot 偵測，會直接以純文字來源提取。
notes 純文字模式已就緒，可直接貼上一般對話內容。
