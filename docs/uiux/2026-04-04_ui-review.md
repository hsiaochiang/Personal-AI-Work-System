# UI Review 2026-04-04

## Review Scope

- 驗收 `import-ui-multi-source` 對 `/extract` 頁面的可見變更
- 檢查重點：
  - 工具來源 selector 是否讓 Import UI 更聚焦，而不是累加更多入口
  - Copilot / ChatGPT / plain 三種來源是否各自顯示對應控制
  - 候選卡片的來源 badge 是否清楚且不壓縮既有審核資訊

## Findings

- `/extract` 的 Step 1 已由疊加式入口改成「先選來源，再顯示對應控制」；這讓 Copilot panel、ChatGPT upload、plain text hint 不再同時爭奪注意力。
- selector 與 per-source panel 仍維持在既有單一卡片內，沒有新增 modal、tab page 或第二條主版面，符合最小安全修改。
- 候選卡片把來源 badge 放在目標檔案列旁邊，與 confidence badge 分工清楚，不會擠壓主要內容文字。
- `plain` 模式預設可見，讓既有手動貼上流程保留最短路徑；Copilot / ChatGPT 模式則以更明確的文案說明各自入口。

## Decision

- UI review PASS
- 本次 UI 收斂直接對齊 V3 brief 的 Import UI scope，且沒有引入新的視覺複雜度或版面跳躍
